עריכת הגדרות

שם החנות יש למלא את שם החנות. תיאור החנות יש למלא את תיאור החנות. לוגו יש למלא את הלוגו. באנר ראשי יש למלא את באנר ראשי. כתובת ראשית (תשמש גם לאיסוף עצמי) יש למלא את כתובת ראשית. קישור אתר אינטרנט (במידה ויש) יש למלא את קישור אתר אינטרנט. קישור לעמוד פייסבוק יש למלא את קישור לעמוד פייסבוק. קישור לפניה בוואצפ קישור לפניה במסנג'ר כתובת דף תשלום יש למלא את כתובת דף תשלום. כתובת עמוד הצלחה יש למלא את כתובת עמוד הצלחה. קוד אנליטיקס קישור לתקנון מטבע יש למלא את מטבע. שפות להצגה - בחירה יש למלא את שפות להצגה.
